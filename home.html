Sobre CutVerse

CutVerse é uma roleta de conteúdo aleatório inspirada na internet dos anos 2000, com uma mistura kawaii, Y2K e nostalgia.

  • Sem login, sem cadastro, sem banco de dados.
  • Conteúdo atualizado a cada clique.
  • Backend em FastAPI, frontend em HTML, CSS e JavaScript puro.

APIs integradas

🐱 CATAAS 🐱 The Cat API 🐕 Dog API 🦊 Random Fox 🦆 Random Duck 🐕 Shibe Online 🐹 Capybara API 😂 Meme API 🎬 Tenor 📸 Picsum

O frontend nunca chama essas APIs diretamente. O backend FastAPI faz a ponte e entrega a surpresa para o CutVerse.
